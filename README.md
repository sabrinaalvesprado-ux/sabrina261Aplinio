# sabrina261Aplinio# ⚽ Projeto Biografia: Neymar Jr.

Este é um projeto de desenvolvimento web focado na criação de uma página biográfica interativa e responsiva sobre o jogador de futebol brasileiro **Neymar Jr.** O site reúne informações históricas, conquistas profissionais, mídias integradas e elementos dinâmicos.

---

## 🎯 Objetivo do Projeto
O principal objetivo deste projeto é consolidar conhecimentos práticos em desenvolvimento front-end (HTML5, CSS3 e JavaScript moderno), aplicando boas práticas de semântica, responsividade e incorporação de mídias externas (como o player do YouTube), enquanto se homenageia uma das maiores figuras do futebol mundial.

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido utilizando a tríade fundamental do desenvolvimento web:
* **HTML5:** Estruturação semântica do conteúdo (tags como `<header>`, `<section>`, `<video-wrapper>`, `<ul>`, etc.).
* **CSS3:** Estilização personalizada, uso de gradientes com as cores temáticas (verde, amarelo e azul), design responsivo e efeitos visuais como sombras e cantos arredondados.
* **JavaScript (ES6+):** Implementação de interatividade e melhorias na experiência do usuário (interações dinâmicas opcionais, controle de eventos ou manipulação simples do DOM).

## 🚀 Funcionalidades
* **Design Totalmente Responsivo:** O layout se adapta perfeitamente a computadores, tablets e smartphones.
* **Seção de Vídeo em Destaque:** Integração fluida e responsiva de um player do YouTube via `<iframe>` sem quebrar a proporção da tela (16:9).
* **Navegação Semântica:** Divisão clara entre a visão geral do atleta, infância, carreira internacional e recordes pela Seleção Brasileira.
* **Identidade Visual Temática:** Cores inspiradas na trajetória do jogador e no Brasil, com contraste otimizado para leitura.
